Pour pouvoir utiliser le fichier de test .side, on vas devoir utiliser l'extension SeleniumIDE:

Chrome --> https://chrome.google.com/webstore/detail/selenium-ide/mooikfkahbdckldjjndioackbalphokd
FireFox --> https://addons.mozilla.org/fr/firefox/addon/selenium-ide/

Une fois installé, cliquez sur son logo en haut à droite de votre nagiteur, plusieurs option s'affiche.
Cliquez sur "open an existing project" et ouvrez le fichier .side

Les tests fonctionneront seulement si le site web est démarrer et lancer avec Local.
